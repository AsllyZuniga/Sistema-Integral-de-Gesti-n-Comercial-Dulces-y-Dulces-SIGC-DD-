info Instrucciones de uso

  1. Exporta el plano de ventas desde el ERP en formato .txt (separado por tabulaciones)
  2. Haz clic en la zona de carga y selecciona el archivo exportado
  3. Verifica que el nombre y el tamaño del archivo sean correctos
  4. Haz clic en Importar Ventas y espera a que finalice el proceso
  5. Revisa el resumen: registros exitosos, errores y tiempo total
warning Para archivos grandes (más de 100 MB) el proceso puede tardar varios minutos. No cierres la ventana mientras se procesa.
