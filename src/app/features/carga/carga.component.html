info Instrucciones de uso

  1. Exporta el plano de ventas desde el ERP en formato .txt o .tsv
  2. Haz clic en la zona de carga y selecciona el archivo exportado
  3. Verifica que el nombre del archivo sea correcto
  4. Haz clic en Importar Ventas y espera a que finalice el proceso
  5. Revisa el resumen de registros importados y errores
warning Para archivos grandes (más de 100 MB) el proceso puede tardar varios minutos. No cierres la ventana.
